# Mapa de Carreira - Victor Mariano

Este projeto apresenta meu mapa de carreira desenvolvido em HTML e CSS, destacando meus objetivos profissionais, habilidades técnicas e competências voltadas para a área de Ciência de Dados.

## Tecnologias utilizadas

- HTML5
- CSS3
- Bootstrap

## Objetivo

Apresentar de forma visual e organizada minha trajetória de desenvolvimento profissional e acadêmico.
